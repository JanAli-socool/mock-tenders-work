# Mini TED Tender Parser MVP

This is a mock implementation of a TED-style tender parser using GPT-mock extraction and Streamlit dashboard.

## Features

- Parses 3 mock tenders
- Extracts structured fields via GPT simulation
- Matches tenders against reference projects
- Displays matches and reasoning
- Provides filtering and CSV export

## Setup

1. Clone repo
2. Install requirements through this : pip install -r requirements.txt
3. Run Streamlit app : streamlit run app.py or python -m streamlit run app.py


## Note

This version uses mock GPT responses and sample reference data for demonstration purposes.
